Нужно сделать форму кредитной карты.

Пример интерфейса: https://drive.google.com/file/d/1CmN_Te5EgcGePq-8nQejh2t83KFTpGdR/view?usp=drivesdk

При изменении полей формы нужно подставлять значения в нужные поня над формой (в саму карточку).

- Поле Card Number должно содержать только цифры и длинна его не должна быть больше или меньше 16 символов.
- Card Name это текстовое поле. Максимальная длинна 10 символов, а минимальная 3.
- CVV должно содержать только цифры и длинна должна быть не больше 3 символа. 

Проверки для полей нужно обязательно сделать через JS, без использования HTML атрибутов.
Кнопка "Submit" должна быть заблокированной, пока пользователь не заполнит все поля.
После нажатия на "Submit" вывести полученные данные в консоль.
